## Getting Started

Run FalkorDB

```bash
docker run -p 6379:6379 -it --rm falkordb/falkordb
```

Install node packages

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
